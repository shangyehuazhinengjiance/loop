# 调整自己发言的展示位置
一、 需求背景与目标
在当前的群聊功能中，所有用户的发言样式和布局保持一致。为了提升用户体验，增强“自我”身份的辨识度，需要针对当前登录用户（即“自己”）的发言进行差异化处理。
核心目标：将当前用户的发言气泡及头像移至屏幕右侧，并应用专属的视觉样式，使其与其他群成员的发言形成明显区分。
二、 详细功能需求
1. 布局调整（左右分布）
当前用户（自己）：
头像：显示在消息气泡的右侧。
气泡：整体靠屏幕右侧对齐。
昵称：隐藏或显示在气泡上方/右侧（视整体UI规范而定，通常自己的消息可隐藏昵称以节省空间，或采用弱化样式）。
其他用户：
头像：显示在消息气泡的左侧（保持现有逻辑不变）。
气泡：整体靠屏幕左侧对齐。
昵称：显示在气泡上方或左侧。
2. 视觉样式定制（特殊样式）
针对当前用户的发言气泡，需应用一套独立的样式配置：
背景颜色：使用专属的主题色（例如：微信绿、品牌主色调等），区别于其他用户的白色/浅灰色气泡。
文字颜色：确保在专属背景色下的文字可读性（通常为白色或深色）。
圆角设计：右侧气泡的圆角方向需与左侧气泡镜像对称（例如：左上、左下、右下圆角，右上角为尖角指向头像）。
特殊标识（可选）：可在气泡角落增加微弱的“我”标识或特殊边框，增强辨识度。
